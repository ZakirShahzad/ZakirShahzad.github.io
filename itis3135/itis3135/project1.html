Overview

Description

This website is for the UNCC Muslim Dawah League. It is an organization that provides faith tables where religious topics are discussed.

Intended Users

The intended users for this website are current members and individuals interested in joining the Muslim Dawah League.

Content Overview

The primary goal of the website is to inform individuals about how to join the Muslim Dawah League, provide updates, and display the charter of the league.
